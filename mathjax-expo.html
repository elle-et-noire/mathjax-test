MathJax 表示例集

ギリシャ文字

  • 大文字立体を出力する $2$ つの方法（\Gamma と \Upgamma など）は同じ出力で、そのようなものは \Gamma などの方のコマンドだけ示しています。
  • 大文字のうちアルファベットで代用できるものはコマンドが存在しません（\Alpha はエラー）。
  • 全角のギリシャ文字は見た目はアルファベットと同じものもありますが、文字コードは異なります。
  • \phi,\varphi は \upphi,\upvarphi になると入れ替わることに注意してください。
  • フォントによっては全角の φ と ϕ が入れ替わって表示されます（例えば筆者の環境では consolas と Yu Gothic UI とでは逆になりました）。
